Pourquoi pas vous ?

Qu'est ce que le déveleppement web ?

Front-end (déveleppement coté client): Ce qui est visible pour l’utilisateur, comme la mise en page, les couleurs, les boutons, les animations, etc.

Back-end (déveleppement coté serveur): Ce qui gère la logique, les bases de données et l'interaction avec les serveurs

Les technologies de base du web

C’est la structure de votre site. HTML permet de définir les éléments sur la page (titres, paragraphes, images, vidéos, etc.).

Exemple

<h1>Bienvenue sur mon site <h1>
<p> Ceci est un paragraphe introductif <p>

CSS s’occupe du style de votre site : couleurs, polices, mise en page, etc. Il permet de rendre votre site attrayant et agréable à utiliser.

Exemple

<h1>Bienvenue sur mon site <h1>
<p> Ceci est un paragraphe introductif <p>

JavaScript est le langage de programmation qui rend votre site interactif. Il permet d’ajouter des animations, des boutons interactifs, des formulaires dynamiques, etc.

Exemple

document.querySelector('h1').addEventListener('click', function() { alert('Vous avez cliqué sur le titre !'); });

Outils essentiels pour débuter

Voici quelques outils importants pour commencer à développer des sites web :

  • Editeur de code: Logiciel pour écrire du code (Exemple : Visual Studio Code).
  • Navigateur: estez vos sites dans différents navigateurs (Chrome, Firefox, etc.).
  • Inspecteur de code : Chaque navigateur a un inspecteur pour voir et tester le code directement sur une page web.
  • GitHub: Un outil pour héberger et collaborer sur des projets de développement.
